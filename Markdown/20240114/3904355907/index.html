Markdown 基礎語法

發表於2024-01-14|更新於2024-01-14|Markdown
|閱讀量:

大家好，這是我的第一篇文章，主要在介紹 markdown 語法

標題

在文字前加上 # 來創建標題。標題等級由 # 的數量決定（從 1 到 6 級）。

# 一級標題

## 二級標題

### 三級標題

#### 四級標題

##### 五級標題

###### 六級標題

段落與換行

直接寫入文本會形成一個段落。在行尾加上兩個或更多的空格然後換行，可以創建一個新的段落。

這是第一段。
這是第二段。

強調（斜體和粗體）

使用 * 或 _ 來創建斜體或粗體文字。

*斜體* 或_斜體_
**粗體** 或 __粗體__

效果如下:
我是斜體
我是粗體

列表

使用 *、- 或 + 創建無序列表，使用數字創建有序列表。

- 項目一
- 項目二
- 項目三

  1. 第一項
  2. 第二項
  3. 第三項

效果如下:

  • 項目一
  • 項目二
  • 項目三
  1. 第一項
  2. 第二項
  3. 第三項

鏈接和圖片

使用 []() 來添加鏈接，使用 ![]() 來添加圖片。
[這是一個鏈接](Link)
![這是一張圖片](Image)

鏈接:
happyRat
圖片:

代碼區塊

使用反引號來表示行內代碼，使用前後三個反引號 ` 創建代碼塊。
```
這是行內代碼：code

```

引用

> 這是一段引用。

這是一段引用。
